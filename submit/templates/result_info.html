제출 결과 설명

✅ 맞았습니다!!

제출한 코드가 옳은 코드입니다.

모든 경우에서, 입력값에 대한 코드의 출력값이 정답 출력과 일치합니다.

코드 제출 시에만 나오는 결과입니다.

❌ 틀렸습니다

제출한 코드가 틀린 코드입니다.

특정한 경우에서, 입력값에 대한 코드의 출력값이 정답 출력과 일치하지 않습니다.

틀린 경우의 입력값과 코드의 출력값, 정답 출력을 화면에서 확인할 수 있습니다.

코드 제출 시에만 나오는 결과입니다.

✅ 실행 완료

테스트 실행이 완료되었습니다.

테스트 실행이 아무런 문제 없이 완료되었습니다.

출력 결과와 소요 시간, 사용한 메모리를 화면에서 확인할 수 있습니다.

테스트 실행 시에만 나오는 결과입니다.

💥 오류 발생

코드 실행 중 오류가 발생했습니다.

특정 경우에서 파이썬 오류 문장이 출력되었습니다.

오류 이름과 오류가 일어난 이유, 오류가 발생한 입력값, 오류가 발생하기 직전까지의 출력값을 화면에서 확인할 수 있습니다.

⏳ 시간 초과

코드 실행이 너무 오래 걸립니다.

특정 경우에서 코드 실행 시간이 5초를 넘어, 서버 과부하 방지를 위해 코드 실행이 중지되었습니다.

시간 초과가 일어난 입력값을 화면에서 확인할 수 있습니다.

💣 메모리 초과

코드가 점유하는 메모리 용량이 너무 큽니다.

특정 경우에서 코드가 사용하는 메모리 용량이 128MB를 넘어, 서버 과부하 방지를 위해 코드 실행이 중지되었습니다.

메모리 초과가 일어난 입력값을 화면에서 확인할 수 있습니다.

📝 출력 초과

코드의 출력값이 너무 깁니다.

특정 경우에서 코드의 출력값 크기가 정답 출력 크기의 2배를 넘어, 서버 과부하 방지를 위해 코드 실행이 중지되었습니다.

테스트인 경우 코드의 출력값이 1000자를 넘으면 이 결과가 발생합니다. 1000자 이상의 출력을 테스트해야 하는 경우 문의를 남겨주세요.

출력 초과가 일어난 입력값을 화면에서 확인할 수 있습니다.

🚩 준비 중

곧 테스트/제출이 진행됩니다.

코드를 실행할 독립된 가상 환경을 생성하고, 입력 데이터를 가상 환경으로 옮기는 중입니다.

이 상태가 10초 이상 지속되면 다시 제출해보세요. 현상이 반복되면 문의에 신고해주세요.

🔁 채점 중

코드 채점이 진행 중입니다.

모든 경우에 대해 코드를 실행하는 중입니다. 각 경우의 순서는 정해져 있으나, 순서를 정하는 기준은 랜덤입니다.

진행률은 정답 출력과 동일한 출력값이 나온 경우의 수, 즉 '맞았습니다!!' 결과가 나온 수를 알려줍니다.

10초 이상 진행률에 변화가 없으면 다시 제출해보세요. 현상이 반복되면 문의에 신고해주세요.

⚠️내부 오류

알 수 없는 오류가 발생하였습니다.

파이가 서버에서 코드를 테스트 또는 채점하던 중, 알 수 없는 오류가 발생하였습니다.

다시 제출해보시고, 현상이 반복되면 좌측 상단 '문의'에 신고해주세요.

이 오류가 발생한 제출은 관리자가 주기적으로 확인합니다 :)
