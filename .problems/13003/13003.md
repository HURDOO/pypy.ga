## 설명
코딩에서 더하기, 빼기, 곱하기, 나누기는 각각 이렇게 사용해요
```python
print(5+2)  # 7
print(5-2)  # 3
print(5*2)  # 10
print(5/2)  # 2.5
```

## 문제
25와 8의 덧셈, 뺄셈, 곱셈, 나눗셈 결과를 출력해보세요.

### 출력값 설명
1. 첫 번째 출력으로, 25와 8의 덧셈 결과를 출력해주세요.
2. 두 번째 출력으로, 25와 8의 뺄셈 결과를 출력해주세요.
3. 세 번째 출력으로, 25와 8의 곱셈 결과를 출력해주세요.
4. 네 번째 출력으로, 25와 8의 나눗셈 결과를 출력해주세요.

### 출력 예제
```
33
17
200
3.125
```
