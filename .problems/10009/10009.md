# 10009: 예체능

## 문제
예체능 과목은 절대평가로, 점수가 80점 이상이면 A, 60점 이상 80점 미만이면 B, 60점 미만이면 C로 구분된다.

학생의 점수가 주어졌을 때, 학생의 등급을 알아보자.

## 입력
첫 번째 줄에 학생의 점수 `N`이 입력됩니다. `(0 ≤ N ≤ 100, N은 정수)`

## 출력
첫 번째 줄에 학생의 등급을 출력합니다.

### 입력 예제 1
```
95
```

### 출력 예제 1
```
A
```

### 입력 예제 2
```
60
```

### 출력 예제 2
```
B
```
