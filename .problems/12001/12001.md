# 12001: 숫자 뒤집기

## 문제
두 자리 자연수 a,b가 입력으로 주어질 때, a와 b를 뒤집은 수를 차례대로 출력하시오.

예를 들어, a=12, b=34라면 2143을 출력한다.

단, 숫자를 뒤집는 함수를 만들어 사용하시오.

## 입력
첫 번째 줄에 a가 입력됩니다.

두 번째 줄에 b가 입력됩니다.

a와 b는 모두 두 자리의 자연수입니다.
## 출력
a와 b를 뒤집은 수를 차례대로 출력합니다. 두 수 사이에 공백은 없습니다.

### 입력 예제 1
```
12
34
```

### 출력 예제 1
```
2143
```

### 입력 예제 2
```
90
16
```

### 출력 예제 2
```
961
```

## 참고
12를 나눈 몫은 1, 나머지는 2이다.

한 자리 숫자에 10을 곱하면 두 자리 수가 된다.
